TEXTRON CESSNA 172 SKYHAWK (G1000)

BEFORE START

  • PARKING BRAKE SET
  • AVIONICS BUSSES OFF
  • IGNITION SWITCH OFF
  • ELECTRICAL SWITCHES OFF
  • MIXTURE IDLE CUT-OFF
  • BAT & ALT MASTER SWITCH ON
  • FUEL QUANTITY CHECK
  • STATIC PRESS ALT SOURCE OFF
    (PUSHED)
  • FUEL SELECTOR VALVE BOTH
  • FUEL SHUTOFF VALVE ON
    (FULL IN)
  • ELEVATOR TRIM SET FOR TO

ENGINE START

  • DOORS CLOSED
  • THROTTLE OPEN 1⁄4 INCH
  • MIXTURE IDLE CUT-OFF
  • PROPELLER AREA CLEAR
  • NAV LIGHT ON
  • BEACON ON
  • FUEL PUMP ON
  • MIXTURE FULL RICH FOR 5 SEC
    VERIFY STABLE FUEL FLOW
  • MIXTURE IDLE CUT-OFF
  • FUEL PUMP OFF
  • IGNITION SWITCH HOLD START
    WHEN ENGINE FIRES
  • IGNITION SWITCH RELEASE
  • MIXTURE FULL RICH
  • OIL PRESSURE VERIFY
  • THROTTLE 1000 RPM

BEFORE TAXI

  • INTERNAL LIGHTS AS RQRD
  • CABIN HEAT AS RQRD
  • AVIONICS BUSSES ON
  • AVIONICS SET
  • ALTIMETER SET
  • TRANSPONDER SET & STBY
  • FUEL QUANTITY CHECK
  • PITOT HEAT AS RQRD
  • FLAPS 10 ̊
  • ELEVATOR TRIM SET FOR TO
  • FLIGHT CONTROLS FREE & CORRECT
  • TAXI LIGHTS ON

TAXI

  • BRAKES CHECK
  • FLIGHT INSTRUMENTS CHECK
  • FLIGHT INSTRUMENTS CHECK

  • MIXTURE FULL RICH
  • FUEL SELECTOR VALVE CHECK BOTH
