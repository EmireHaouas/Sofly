  • FLIGHT CONTROLS FREE & CORRECT
  • PARKING BRAKE SET
  • POWER LEVER IDLE DETENT
  • IDLE CONTROL LEVER CUT-OFF
  • PROPELLER CONTROL LEVER FEATHER IF OAT > +10°C
    FEATHER IF OAT > +10°C
    FULL FORWARD IF OAT < +10°C
  • STARTER SWITCH OFF
  • IGNITION SWITCH OFF
  • GENERATOR SWITCH OFF
  • BOOSTER PUMP SWITCH OFF
  • LDG LIGHTS OFF
  • RADIO MASTER OFF
  • MASTER SWITCH BATTERY
    CHECK VOLTAGE
  • ROTATING BEACON ON
  • FUEL SYSTEM VALVE OPEN, GATED
  • FUEL QTY/TOTALIZER CHECK / SET
  • ENGINE INSTRUMENTS CHECK

ENGINE START

  • BOOSTER SWITCH ON
  • PROP AREA CLEAR
  • STARTER SWITCH ON
  • OIL PRESSURE CHECK RISING
  • IGNITION SWITCH ON
    WHEN NG STABILIZED (MIN 12% NG)
  • IDLE CONTROL LEVER LOW IDLE
  • ITT MONITOR
    (MAX 1090°C FOR 2 SECONDS)
    WHEN LOW-IDLE RPM IS ATTAINED
    (MIN 46% NG)
  • STARTER SWITCH OFF
  • IGNITION SWITCH OFF
  • OIL PRESSURE CHECK GREEN ARC
  • ITT STABILIZED BELOW 660°C

AFTER START

  • PROPELLER CONTROL LEVER FWD
  • GENERATOR SWITCH ON
  • NG CHECK ABOVE 51%
  • RADIO MASTER ON
  • AVIONICS ON / SET AS RQRD
  • COMPASS SYNCHRONIZED
  • ARTIFICIAL HORIZONS ERECT
  • NAV LIGHTS ON

TAXI

  • PARK BRAKE RELEASE
  • BRAKES CHECK
  • FLIGHT INSTRUMENTS CHECK

BEFORE TAKE OFF

  • ALL TRIMS GREEN MARK
  • FLAPS 28°
  • FLIGHT CONTROLS FREE & CORRECT
  • ALTIMETER SET
  • FUEL QUANTITY CHECK
  • BOOSTER PUMP SWITCH ON
  • PITOT HEAT AS RQRD
  • OIL TEMP GREEN ARC
  • INSTRUMENTS CHECKED
  • DOORS/WINDOWS CLOSED
  • LANDING LIGHTS DOWN AND ON

TAKE OFF

  • IDLE CONTROL LEVER HIGH IDLE
  • THROTTLE FULL

CLIMB

  • FLAPS UP
  • BOOSTER PUMP SWITCH OFF
  • LANDING LIGHT(S) UP/OFF
  • HEATING AS RQRD
  • ITT MAX 695°C
