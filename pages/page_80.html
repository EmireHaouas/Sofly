TEXTRON CESSNA CITATION LONGITUDE

PREFLIGHT

  • CIRCUIT BREAKERS IN
  • STBY PWR SWITCH TEST/ON
  • BATT BUTTONS ON, CHECK VOLTS
  • EIS/CAS CHECK
  • APU
    WHEN APU HAS STARTED
    START
  • APU GEN RESET/ON
  • EMER LTS SWITCH ARM
  • TRIMS SET
  • FUEL QUANTITY CHECK
  • AUTOPILOT ENGAGE/DISENGAGE

BEFORE START

  • PARKING BRAKE SET
  • CABIN DOOR CLOSED
  • SEAT BELTS BUTTON ON
  • PAX SAFETY BUTTON ON
  • EIS/CAS CHECK
  • RECOG LIGHTS ON

ENGINE START

  • ANTI COLL LIGHTS ON
  • THROTTLES IDLE
  • R ENG RUN/STOP RUN
  • R ENG START BUTTON PRESS
  • R ENG INSTRUMENTS CHECK
  • L ENG RUN/STOP RUN
  • L ENG START BUTTON PRESS
  • L ENG INSTRUMENT CHECK
  • L/R GEN RESET/ON
  • BUS TIE OPEN
  • APU GEN OFF

BEFORE TAXI

  • FLIGHT CONTROLS FREE & CORRECT
  • APU OFF
  • SPEEDBRAKES CHECK/RETRACTED
  • FLAPS 2
  • AVIONICS CHECK AND SET
  • ICE PROTECTION AS RQRD
  • TAXI LIGHT ON

BEFORE TAKE OFF

  • FLAPS CHECK 2
  • SPEEDBRAKES CHECK RETRACTED
  • TRIMS CHECK SET
  • ICE PROTECTION CHECK, AS RQRD
  • V SPEEDS DISPLAYED
  • INSTRUMENTS SET
  • WEATHER RADAR AS RQRD
  • LANDING LIGHTS ON

AFTER TAKE OFF

  • LANDING GEAR UP
    PASSING V2 +20
  • FLAPS RETRACT
  • THROTTLES CLB
    PASSING 10.000 FEET
  • PRESSURISATION CHECK
  • SEATBELTS BUTTON OFF
  • ALTIMETERS CHECK SET
  • LIGHTS AS RQRD
