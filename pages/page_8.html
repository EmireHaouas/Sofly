BOEING B747-8I

PRE-FLIGHT

  • PARKING BRAKE ON
  • TROTTLE IDLE
  • GEAR LEVER CHECK DOWN
  • FLAPS UP
  • SPEED BRAKE RETRACTED
  • BATTERY SWITCH ON
  • STANDBY POWER SWITCH AUTO
  • ADIRS ALL NAV
  • FUEL CONTROL SWITCHES CUTOFF
  • NAV & LOGO LIGHT ON
  • APU AS RQRD
  • FLIGHT INSTRUMENTS SET
  • ALTIMETERS __SET
  • FUEWL QUANTITY SETUP

BEFORE START

  • FUEL PUMPS ON/AS RQRD
  • SEAT BELTS SIGNS ON
  • WINDOWS/DOORS CEHCK CLOSED
  • ENGINE BLEEDS SET
  • APU BLEED SET
  • RECALL CHECKED
  • BEACON LIGHT ON/LWR
  • TRANSPONDER SET/STBY

AFTER START

  • APU OFF
  • ANTI ICE AS RQRD
  • REF SPEEDS CHECKED
  • RECALL CHECKED
  • AUTOBRAKES RTO
  • GROUND EQUIPMENT CLEAR
  • FLAPS __SET
  • FLIGHT CONTROLS CHECKED
  • TAXI/RWY LIGHTS ON
  • PARKING BRAKE OFF

BEFORE TAKE OFF

  • DEPARTURE BRIEFING COMPLETE
  • CABIN READY
  • LANDING LIGHTS ON
  • STROBE LIGHTS ON
  • TRANSPONDER TA/RA
  • WEATHER RADAR AS RQRD

AFTER TAKE OFF

  • LANDING GEAR UP
  • TAXI/RWY LIGHTS OFF
  • FLAPS UP
  • ALTIMETERS STD SET
