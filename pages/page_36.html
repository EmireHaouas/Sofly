HUGHES HERCULES H4

BEFORE START

  • ALL LIGHTS OFF
  • MIXTURES IDLE CUTOFF
  • MASTER THROTTLES CLOSED
  • WING FUEL TANKS OPEN
  • HULL FUEL TANKS OPEN
  • BATTERY ON
  • APU GENERATORS (1,2) ON
  • APU STARTERS (1,2) ON

ENGINE START

  • COWL FLAPS OPEN
    START SEQUENCE: 5-6, 3-4, 7-8, 1-2
  • IGNITION ON
  • MIXTURE AUTO RICH
  • STARTER ENGAGE
  • PRIME ON

WARM UP

  • FLIGHT CONTROLS CHECKED
  • PROPS EXERCISED
  • ENGINE RUN UP COMPLETE

TAXI

  • ALTIMETER SET
  • LANDING LIGHTS ON
  • ENGINE GENERATOR (1,3,7) ON

TAKE OFF

  • WINDOWS & HATCHES SECURE
  • FLAPS 20°
  • ELEVATOR TRIM 0°
  • ALL EXTERIOR LIGHTS ON
  • PITOT HEAT AS RQRD
  • T/O POWER (49.5” 2550 RPM) SET

CLIMB

  • CLIMB POWER (43” 2550 RPM) SET
    @ SAFE ALTITUDE
  • FLAPS UP
  • APU (1,2) OFF
  • APU GENERATORS (1,2) OFF
  • COWL FLAPS AS RQRD

CRUISE

  • LANDING LIGHTS OFF
  • CRUISE POWER AS RQRD

DESCENT

  • PITOT HEAT AS RQRD
  • DESCENT POWER MONITOR

LANDING

  • FLAPS LOWER IN 10° INCREMENTS
  • LANDING LIGHT ON
  • AUTOPILOT OFF

AFTER LANDING

  • FLAPS UP
  • COWL FLAPS OPEN
  • APU (1,2) ON
  • APU GENERATORS (1,2) ON
  • ENGINE GENERATOR (1,3,7) OFF

PARKING

  • MIXTURES IDLE CUTOFF
  • IGNITIONS OFF
  • MASTER THROTTLES CLOSED
  • COWL FLAPS CLOSED, NEUTRAL
  • ALL LIGHTS OFF
  • BATTERY OFF
  • APU GENERATORS (1,2) OFF
  • APU (1,2) OFF
  • WING FUEL TANKS CLOSED
  • HULL FUEL TANKS CLOSED
