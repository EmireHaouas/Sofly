GRUMMAN G-21A GOOSE

BEFORE START

  • BATTERY ON
  • GENERATOR ON
  • VOLT BUTTON CHECK
  • FUEL VALVES OPEN
  • PROP LEVERS FULL FORWARD
  • MIXTURE LEVERS FULL RICH
  • THROTTLES CLOSED
  • PARKING BRAKE ON

ENGINE START

  • BEACON LIGHT ON
  • RIGHT MAGNETO BOTH
  • WOBBLE FUEL PUMP APPLY
    (BLACK LEVER BEHIND LEFT SEAT)
  • FUEL PRESSURE CHECK
  • RIGHT ENGINE START BUTTON PUSH
  • RIGHT THROTTLE APPLY
  • RPM AND MAN PRESSURE CHECK
  • LEFT MAGNETO BOTH
  • LEFT ENGINE START BUTTON PUSH
  • LEFT THROTTLE APPLY
  • RPM AND MAN PRESSURE CHECK

BEFORE TAXI

  • PITOT HEAT AS RQRD
  • NAV LIGHTS ON
  • FLIGHT CONTROLS CHECK
  • TAIL WHEEL LOCK OFF

TAXI

  • PARKING BRAKE OFF
  • BRAKES CHECK
  • FLIGHT INSTRUMENTS CHECK

BEFORE TAKE OFF

  • FLAPS CHECK UP
  • PITOT HEAT AS RQRD
  • LANDING LIGHT ON

TAKE OFF

  • TAIL WHEEL LOCK ON
  • THROTTLES FULL
  • 80 MPH ROTATE
  • GEAR UP

CRUISE

  • LANDING LIGHT OFF
  • THROTTLES 27“
  • PROP 1900RPM
  • ENGINE INSTRUMENTS CHECK

APPROACH

  • LANDING LIGHT ON
  • ALTIMETER SET
  • PROP LEVERS FULL FORWARD
  • SPEED 100 MPH
  • FLAPS 30 ̊

LANDING

    WATER
  • GEAR UP
  • SPEED 75 MPH
  • FLAPS 60 ̊
    NON WATER
  • GEAR DOWN
  • SPEED 75 MPH
  • FLAPS 60 ̊

AFTER LANDING

  • FLAPS UP
  • LANDING LIGHT OFF
  • TAIL WHEEL LOCK OFF
  • PITOT HEAT OFF

PARKING

  • PARKING BRAKE ON
  • FUEL VALVES OFF
  • FUEL PRESSURE CHECK
  • THROTTLES CLOSED
  • NAV LIGHTS OFF
  • BEACON OFF
  • MAGS OFF
  • GENERATOR OFF
  • BATTERY OFF

Not to be used for real-world navigation or aviation purposes.
