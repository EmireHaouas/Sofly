TEXTRON CESSNA 152 (AEROBAT)

BEFORE START

  • PARKING BRAKE SET
  • FUEL SHUT OFF VALVE ON
    (OPEN)
  • ELECTRICAL EQUIPMENT OFF
  • CIRCUIT BRAKERS CHECK IN

ENGINE START

  • MASTER SWITCH (BAT&ALT) ON
  • NAV LIGHTS ON
  • BEACON ON
  • MIXTURE FULL RICH
  • CARB HEAT COLD
    (PUSHED)
  • PRIMER AS RQRD
    (UP TO 3 STROKES)
  • THROTTLE OPEN 1/2 INCH
  • PROPELLER AREA CHECK CLEAR
  • IGNITION SWITCH START
    (RELEASE WHEN ENGINE FIRES)
  • THROTTLE 1000 RPM
  • OIL PRESSURE CHECK
  • AMMETER CHECK

BEFORE TAXI

  • RADIOS SET
  • TRANSPONDER STBY
  • ALTIMETERS SET
  • FLIGHT CONTROLS FREE & CORRECT
  • DOORS CLOSED & LATCHED
  • TAXI LIGHTS ON

TAXI

  • PARKING BRAKE RELEASE
  • BRAKES CHECK
  • FLIGHT INSTRUMENTS CHECK
  • ENGINE INSTRUMENTS CHECK

RUN-UP

  • PARKING BRAKE SET
  • MIXTURE FULL RICH
  • THROTTLE 1700 RPM
  • MAGNETO LEFT, BOTH, RIGHT, BOTH
    MAX DROP 125 RPM; MAX DIFF 50 RPM
  • CARB HEAT CHECK ON, THEN OFF
    CHECK FOR RPM DROP
  • AMMETER CHECK POSITIVE LOAD
  • SUCTION GAUGE GREEN RANGE
  • OIL TEMP & PRESS GREEN RANGE
  • THROTTLE FULL OPEN
    CHECK FOR 2280 – 2380 RPM
  • THROTTLE 1000 RPM

BEFORE TAKE OFF

  • PARKING BRAKE SET
  • TAXI LIGHT OFF
  • FLIGHT INSTRUMENTS CHECK
  • ENGINE INSRUMENTS CHECK
  • RADIOS SET
  • TRANSPONDER ALT
  • ALTIMETERS SET
  • PITOT HEAT AS RQRD
  • CARB HEAT OFF
  • MIXTURE FULL RICH
  • FUEL SHUTOFF VALVE ON (OPEN)
  • ELEVATOR TRIM SET TAKEOFF
  • FLAPS 10 ̊
  • LANDING LIGHT ON
  • STROBES ON

TAKE OFF

  • TOE BRAKES HOLD
  • THROTTLE FULL
  • TOE BRAKES RELEASE
  • 50 KIAS ROTATE
  • CLIMB 55 KIAS
    @ SAFE ALTITUDE
  • SPEED 75 KIAS
    > 60 KIAS
  • FLAPS 0 ̊

Not to be used for real-world navigation or aviation purposes.
