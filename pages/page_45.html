ROBIN DR400-11 CADET

BEFORE START

  • BATTERIE ON
  • ALARM PANEL TEST;
    SET JOUR(DAY)/NUIT(NIGHT)
  • FUEL VALVE OUVERT
  • FUEL QUANTITY CHECK
  • ANTICOL LIGHTS ON

ENGINE START

  • CARB HEAT (RECH CARBU) COLD
    (PUSHED)
  • MIXTURE FULL RICH
    (UP)
  • FUEL PUMP ON
    (WHITE SWITCH NEXT TO FUEL PRESS)
  • MAGNETOS BOTH
  • THROTTLE 1 CM
  • STARTER BUTTON PRESS
    (RED BUTTON BLW FUEL SELECTOR)
  • THROTTLE 1000 RPM
  • OIL PRESSURE CHECK IN GREEN

AFTER START

  • THROTTLE 1200 RPM
  • OIL PRESSURE CHECK IN GREEN
  • FUEL PUMP OFF
  • FUEL PRESSURE CHECK IN GREEN
  • ALT SWITCH ON
  • VOLTMETER CHECK IN GREEN
  • ALARM PANEL CHECK ALL OFF
  • RADIO SWITCH ON
  • DIRECTIONAL GYRO SET
  • TRANSPONDER SBY
  • ALTIMETER SET

TAXI

  • TAXI LIGHTS (ROULAGE) ON
  • NAV LIGHTS (NAVIGAT) ON
  • PARKING BRAKES UNLOCKED
  • BRAKES & RUDDER TEST
  • INSTRUMENTS CHECK
  • THROTTLE 1200 RPM

RUN-UP

  • PARKING BRAKES ON
  • TAXI LIGHTS OFF
  • ENGINE INSTRUMENTS GREEN
  • MIXTURE FULL RICH
  • CARB HEAT COLD
    (PUSHED)
  • THROTTLE 2000 RPM
  • MAGNETOS L, BOTH, R, BOTH
    MAX DROP 150RPM; MAX DIFF 50RPM
  • CARB HEAT HOT
    (PULLED)
    CHECK DROP OF 100 RPM
  • THROTTLE IDLE
    CHECK 600-650 RPM
  • CARB HEAT COLD
    (PUSHED)
  • THROTTLE 1200 RPM

BEFORE TAKE OFF

  • FLIGHT CONTROLS FREE & CORRECT
  • FLAPS FIRST NOTCH
  • ELEVATOR TRIM T/O POSITION (~4)
  • MIXTURE RICH(FULL UP)
  • FUEL PRESSURE CHECK
  • FUEL PUMP ON
  • CARB HEAT COL
  • MAGNETOS BOTH
  • ENGINE INSTRUMENTS GREEN
  • ALARM PANEL CHECK ALL OFF
  • TRANSPONDER ALT
  • DIRECTIONAL GYRO SET
  • LANDING LIGHTS (ATTERIS) ON
  • PARKING BRAKES RELEASE
