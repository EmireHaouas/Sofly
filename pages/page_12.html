CUBCRAFTERS NXCUB

Before start

  • FLIGHT CONTROLS FREE & CORRECT
  • MASTER SWITCH ON
  • TRIM CHECK AND SET FOR TAKEOFF
  • FUEL SELECTOR FULLEST TANK
  • FUEL QUANTITY CHECK
    (ON THE INSIDE OF THE WINGS)
  • MIXTURE IDLE CUTOFF

BEFORE START

  • PARKING BREAKES SET
  • FUEL SELECTOR FULLEST TANK
  • AVIONICS MASTER SWITCH OFF
  • NAV & STROBE LIGHTS ON

ENGINE START

  • MASTER SWITCH CHECK ON
  • PROP LEVER FULL FORWARD
  • MIXTURE LEVER FULL RICH
  • THROTTLE FULL FORWARD
  • FUEL PUMP ON, CHECK FOR FUEL FLOW
  • FUEL PUMP OFF
  • THROTTLE 1/2 INCH OPEN
  • PROP AREA CLEAR
  • STARTER KEY START
    (RELEASE WHEN ENGINE FIRES)

AFTER START

  • OIL PRESSURE CHECK
  • THROTTLE 1000 RPM
  • ALTERNATOR FIELD SWITCH ON
  • AVIONICS MASTER SWITCH ON
  • ALTIMETER SET

TAXI

  • PARKING BRAKES OFF
  • TOE BRAKES CHECK
  • FLIGHT INSTRUMENTS CHECK

BEFORE TAKE OFF

  • PARKING BRAKES SET
  • THROTTLE 1700 RPM
  • IGNITION L, BOTH, R, BOTH
  • ENGINE INSTRUMENTS CHECK
  • PROP EXERCISE TWICE, THEN
    FULL FORWARD
  • THROTTLE CHECK IDLE
  • THROTTLE 1000 RPM
  • FLIGHT INSTRUMENTS CHECK
  • FUEL SELECTOR FULLEST TANK
  • PROP FULL FORWARD
  • MIXTURE FULL RICH
  • TRIM SET
  • FLAPS FIRST NOTCH
  • FLIGHT CONTROLS FREE & CORRECT
  • TRANSPONDER ON & SET
  • LANDING LIGHTS ON
  • IGNITION CHECK BOTH
  • PARKINNG BRAKE RELEASE

TAKE OFF

  • THROTTLES FULL
  • 50 KIAS ROTATE
  • CLIMB 60 KIAS
    @ SAFE ALTITUDE
  • FLAPS RETRACT

CLIMB

  • PROP 2600RPM
  • MIXTURE LEAN FOR MAX RPM
  • LANDING LIGHTS OFF

Not to be used for real-world navigation or aviation purposes.
